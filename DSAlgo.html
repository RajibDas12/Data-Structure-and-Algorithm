Asymptotic Notation :

Asymptotic notations gives us an idea about how good a given algorithm is compared to some other algorithms.

Mathematical Definition of "order of" now.

Primarily there are three types of widely used asymptotic nonations.

  1. Big oh nonation (o)
  2. Big omega notation
  3. Big theta notation

Big oh notation :

Big oh notation is used to describe asymptotic upper bound.

Mathematically, if f(n) describes running time of an alforithm; f(n) is O(g(n)) if there exist positive constants C and n0 such that

O <= f(n) <= c g(n) ......................for all n >= n0

Big omega notation :

Just like O notation provides an asymptotic uper bound, omega notation provides asymtotic lower bound. Let f(n) define running time of an algorithms.

f(n) is said to be omega(g n) if there exists positive constants C and n0 such that

O <= c g(n) <= f(n).......................for all n >= n0

Big theta notation :

Let f(n) define running time of an algorithms.

f(n) is said to be theta(g n) if f(n) is O(g(n)) and f(n) is theta(g(n))

Mathematically,

O <= f(n) <= c1 g(n)

O <= c2 g(n) <= f(n)

Merging both the equations, we get :

O <= c2 g(n) <= f(n) <= c1 g(n)

The equation simply means there exist positive constants c1 and c2 such that f(n) is sanduiched between c2 g(n) and c1 g(n).

Which one of these to use ?

Since Big theta gives a better picture of runtime for a given algorithm.Most of the interviewers expect you to provide an answer in terms of Big theta when they say "order of".
